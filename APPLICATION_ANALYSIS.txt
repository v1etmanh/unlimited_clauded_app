5. HE THONG LICNESE (LICENSE SYSTEM)
================================================================

Kieu bao mat: Asymmetric-like (server tao token, client chi validate)

Luong:
  1. Admin tao key: keygen.py --email x --days 30
     -> Key format: XXXXX-XXXXX-XXXXX-XXXXX-CCCC
     -> Luu vao licenses.json (tren server)
     -> HMAC-SHA256 signature voi SECRET_KEY

  2. User nhap key vao app
     -> App gui key len license_server.py
     -> Server tra ve token + expiry info
     -> App luu cache (Base64 obfuscated)

  3. Moi 24h: background thread ping server
     -> Kiem tra key co bi revoke khong
     -> Neu bi revoke: thoat app

  4. Offline fallback:
     -> Neu khong co internet: dung cache cu
     -> Cache co local date check

Ban do bao mat:
  Attack               | Defense
  ---------------------|----------------------------
  Share key            | 24h revoke check
  Fake key             | HMAC-SHA256 signature
  Tamper cache         | Base64 obfuscation
  Use after expiry     | Server + local date check
  Bypass completely    | PyInstaller obfuscation

7. API ENDPOINTS
================================================================

  Method | Path                    | Mo ta
  -------|-------------------------|---------------------------
  GET    | /health                 | Kiem tra trang thai server
  GET    | /v1/models              | Danh sach model
  POST   | /v1/chat/completions    | Gui tin nhan (stream/non)

Request body (chat completions):
  {
    "model": "claude-via-browser",
    "messages": [
      {"role": "system",    "content": "..."},
      {"role": "user",      "content": "..."},
      {"role": "assistant", "content": "..."}
    ],
    "stream": false
  }

Response: OpenAI-compatible format
  {
    "id": "chatcmpl-...",
    "object": "chat.completion",
    "choices": [{"message": {"role": "assistant", "content": "..."}}]
  }

10. DEPLOYMENT
================================================================

Development:
  python app_with_license.py

Build .exe:
  pyinstaller --onefile --noconsole \
    --name "unlimited_claude_api" app_with_license.py
  -> Output: dist/unlimited_claude_api.exe

License Server (Railway/Render):
  - Deploy: license_server.py + licenses.json
  - Env var: PORT=8080
  - Update LICENSE_SERVER url trong license_validator.py

Multi-OS Build:
  - Windows : pyinstaller tren Windows -> .exe
  - macOS   : pyinstaller tren macOS   -> .app / binary
  - Linux   : pyinstaller tren Linux   -> ELF binary
  (Moi OS phai build rieng tren OS do)
